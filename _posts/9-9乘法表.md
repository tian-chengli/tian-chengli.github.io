####  9*9乘法表

```html
<style>
ul li{
    list-style: none;
}
ul li span{
    display: inline-block;
    width: 100px;
    height: 40px;
    line-height: 40px;
    border:1px solid #333;
    text-align: center;}
    </style>
</head>
<body>
<ul id="list"></ul>
<script type="text/javascript">
var oUl=document.getElementById("list");
var str="";
for (var i = 1; i<=9; i++) {
str +="<li>";
    for (var j = 1; j <=i; j++) {
str+=j+"*"+i+"="+j*i+"&nbsp;";

    }
    str+="</li>"
}
oUl.innerHTML=str;
</script>

```
